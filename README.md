# votesavvy

A resource for young voters to explore voting options, through data and social media.

![Sequence Chart](./design/votesavvy-sequence.png)

Need and independent account for:

- [x] Gmail
- [x] Twitter API (they already have twitter so it should be fine)
- [x] Bluemix
- [x] Typeform
- [x] Github
- [x] Zapier

Current features of app

1) users' sign in with twitter

2) users' session management with mongodb

3) survey data stored in cloudant db

4) caching with datacache for the used fields contained in the responses of requests to the represent

5) logging with mongodb
